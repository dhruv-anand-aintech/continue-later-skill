HOW TO SUBMIT TO SKILLFISH:

1. Create GitHub Repository
   
   git remote add origin https://github.com/YOUR_USERNAME/continue-later-skill
   git push -u origin main

2. Go to Skillfish Submission

   https://mcpmarket.com/submit?type=skill

3. Fill in the Form

   - Skill Name:     Continue Later
   - Repository:     https://github.com/YOUR_USERNAME/continue-later-skill
   - Description:    Generate and manage project continuation documentation 
                     for seamless handoffs and knowledge preservation
   - Category:       Productivity or Documentation
   - Keywords:       continuation, handoff, documentation, team-workflow

4. Submit!

   Skillfish team will review and approve

5. Skill Goes Live

   - Appears on Skillfish marketplace
   - Available to all Cursor users
   - Downloads tracked automatically
   - Users can rate and review

WHAT THE SKILL DOES:

    Generate Phase (continue-later):
    - Input: YAML configuration file
    - Output: Beautiful continuation.md file
    - Use: Save project state, hand off to team, document decisions

    Resume Phase (resume-from-earlier):
    - Input: continuation.md file
    - Output: Full file or specific section
    - Use: Understand project state, resume after break
